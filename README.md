# Simple Chat Simulator

This is a desktop chat simulator built in Python using Tkinter.  
Users can send messages, which can include timestamps, emojis, bold, and italic text.  
The project demonstrates design patterns such as Factory, Builder, Decorator, Observer, and Singleton.

## How to Run
Open a terminal in the src/ folder and run:

python main.py

## Dependencies
Python 3.8 or higher (Tkinter is built-in)
